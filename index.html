InView Highlighter

An extension for chromium-based browsers (Chrome, Brave, Edge, and other) that allows to highlight text on web pages and export highlights as JSON (reverse import possible) or HTML.

Features

Use Mobirise website building software to create multiple sites for commercial and non-profit projects.

Highlight with one action

Select text and highlight it instantly via right-click or a keyboard shortcut. Highlights are stored locally for the current page and re-applied when you refresh or revisit.

Export/Import

Export all highlights from the current page to a single JSON file for backup and reuse or to a standalone HTML file containing all highlighted snippets with basic formatting preserved (bold, italics, links).

Clear highlights

Remove all highlights for the current page (both the visible marks and the stored data).

How it works

Highlight

Select text on any page and highlight it using a handy and customisible shortcut Alt+H or the context menu.

Reload-proof

The extension stores the highlight locally so they will always appear until they are cleared (by default Alt+C).

HTML Export

Generate a standalone HTML file containing all highlighted snippets with basic formatting (bold, italics, links).

Backup & share

Export current highlights to a single JSON file to share with others (and your future self!)

Privacy

This extension stores your highlights locally in your browser (chrome.storage.local). It does not send data to any server, does not require an account, and does not track you.

Frequently Asked Questions

Which browsers are supported?

Chromium-based browsers that support Manifest V3 (Chrome v. 126+, Edge April 2021 and later, Brave v. 1.46.144+, etc.).

Why does it not work on some pages?

Browsers block extensions on certain URLs such as chrome:// pages and the Chrome Web Store.

Do keyboard shortcuts work out of the box?

Most of thee times.
If a shortcut conflicts with your system or browser, assign it manually in chrome://extensions/shortcuts.

Will the HTML export keep the exact look of the page?

It preserves basic markup inside the selection (links, bold/italics when they are real HTML tags). If styling is only CSS-based, the exported snippet may not look identical.

Can I import highlights onto a different page?

Imports are intended for the same page URL. If the content differs, highlights may not anchor correctly.

How do I remove all stored data?

Use “Clear highlights” on each page, or remove the extension’s data via your browser’s extension settings.
